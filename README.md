# Back End - RESTful API (El Tiempo)
Proyecto de Prueba Técnica para el periodico El Tiempo de Colombia

## Summary
Creación de RESTful API utilizando NodeJS, Express
- Rutas Protegidas
- Password hashing
- CRUD
- Express
- MongoDB
- Mensajes de Error
- Prueba Unitarias
- Mucho Más

## Ejemplo variables ENV
- MONGO_URI=
- JWT_SECRET=
- FRONTEND_URL=
- PORT=

## FrontEnd:
Realizado en React + Vite
